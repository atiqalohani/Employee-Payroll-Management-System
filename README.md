# Employee Payroll Management System — C++ OOP

1. Download or clone the repository and open `EmployeePayrollSystem.cpp` in Dev C++ or VS.
2. Go to Tools → Compiler Options** and make sure TDM-GCC (MinGW) is selected as the compiler.
3. Press *F11* (or go to *Execute → Compile & Run*) to build and launch the program.
4. On the welcome screen, choose 
  *[1] Sign Up* to create an account, then 
  *[2] Sign In* to log in and access the main menu.
5. Use the numbered menu options to add employees, view records, generate payroll reports, or run the built-in OOP demonstrations.
